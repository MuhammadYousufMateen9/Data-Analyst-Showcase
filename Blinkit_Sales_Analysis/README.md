# Blinkit Sales Analysis Dashboard

## Overview
This project analyzes Blinkit grocery sales data to uncover insights about products, outlets, and customer ratings. 
The dataset contains details such as item type, outlet type, outlet size, sales, and ratings. 
The analysis was performed in Excel using Pivot Tables, Charts, Slicers, and Timelines, and summarized into an interactive dashboard.

## Dataset
- **Source:** Blinkit Grocery Data (Excel file)
- **Key Columns:**
  - Item details: Identifier, Type, Weight, Fat Content, Visibility
  - Outlet details: Year of Establishment, Location Type, Identifier, Size, Outlet Type
  - Performance metrics: Sales, Rating

## Dashboard Features
- **KPIs**: Total Sales, Average Sales, Average Rating, Item Distribution
- **Filters**: Implemented with Slicers for item attributes and Timelines for date fields
- **Visuals**: Sales trends, Item type contribution, Outlet performance

## Tools Used
- Microsoft Excel
  - Pivot Tables & Charts
  - Slicers
  - Dashboard design

## Insights
- Supermarket Type1 contributes the highest overall sales.
- Medium-sized outlets have stronger sales than small ones.
- Sales are highest in Tier 3 locations, showing strong urban demand.
- Rating distribution indicates customer satisfaction with most products.

## How to Use
1. Open the Excel file: **Blinkit_Sales_Analysis.xlsx**
2. Explore pivot tables in the `Pivot Tables & Charts` sheet
3. Interact with the `Dashboards` sheet using slicers
